
   /----COMANDO-----/

 ls :se muestra una lista de información sobre archivos.
 aptitude es una función de gestión de paquetes disponible en algunas versiones de Linux.
 pwd :imprime el directorio de trabajo yindica que el usuario está actualmente en su carpeta de inicio.
 cd :para cambiar de directorio y navegar por la estructura del sistema de archivos.
 exit:Limpia la pantalla.
 Su:le permite actuar temporalmente como un usuario diferente.
 sudo: sólo proporciona acceso administrativo para la ejecución del comando especificado.
 chmod:se utiliza para cambiar los permisos de un archivo o directorio. 
 Cat:se utiliza para ver rápidamente el contenido de archivos pequeños.
 more:Estos y otros comandos adicionales usados para visualizar archivos. 
 less:Estos y otros comandos adicionales usados para visualizar archivos .
 cp :se utiliza para copiar archivos. Similar al comando mv.
 head:(a-j)se utilizan para ver un número seleccionado de líneas desde la parte superior o inferior de un archivo.
 tail:(q-z)se utilizan para ver un número seleccionado de líneas desde la parte superior o inferior de un archivo.
 dd :se utiliza para copiar archivos o particiones enteras al nivel de bits.
